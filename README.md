# 🗓️ WeekMePro Landing Page (Astro + Tailwind)

This is the marketing site for **WeekMePro**, a React 19 SaaS for weekly scheduling. It is built with **Astro** to maximize SEO and speed, serving as the funnel to the main PWA.

---

## 🤖 Claude Agent Instructions (/init)

When you initialize this project, follow these core principles:

### 🎯 Mission
Build a high-conversion, SEO-optimized landing page that targets:
1. **Teachers** (Lesson planning)
2. **Students** (Class schedules)
3. **Gyms/Coaches** (Session booking/routines)

### 🛠 Tech Stack
- **Framework:** Astro (Static Site Generation)
- **Styling:** TailwindCSS 4 (Utility-first, rounded-2xl, clean)
- **State:** Zero-JS by default (Keep it static for speed)
- **Structure:** `src/layouts/`, `src/components/sections/`, `src/pages/`

### 🎨 Visual & Copy Style
- **Tone:** Professional, witty, and empathetic. 
- **Design:** "SaaS-sleek". Plenty of whitespace, vibrant accents on white/soft-gray backgrounds.
- **Copy:** Use active verbs (e.g., "Plan in seconds" instead of "Easy planning").

### 🌍 Multi-language (i18n)
Prepare the site for **English (EN)**, **Spanish (ES)**, and **Catalan (CA)** using Astro's routing patterns.

### 📜 Development Rules
1. **SEO First:** Every component must include proper metadata (Title, Description, OG tags).
2. **PWA Promotion:** Mention the app is installable and works offline.
3. **Call to Action:** The main CTA must point to `https://weekmepro.com/app` (or your app URL).

---

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
